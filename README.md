#  Python Alarm Clock

Um despertador simples feito em **Python**, com interface gráfica usando **Tkinter**.  
O usuário pode definir horas, minutos e segundos, e o alarme tocará no horário escolhido.

---

#  Funcionalidades

- Interface gráfica simples e intuitiva
- Seleção de **hora, minuto e segundo**
- Verificação automática do horário atual
- Emite som quando o horário do alarme é atingido
- Compatível com Windows (usa `winsound`)

---

#  Tecnologias utilizadas

- Python 3
- Tkinter
- datetime
- time
- winsound (Windows)

---

# Pré-requisitos

- Python 3 instalado
- Sistema operacional **Windows**
- Arquivo de áudio `alarm.wav` na mesma pasta do projeto

---


#  Como executar

1. Certifique-se de ter o Python instalado
2. Baixe os arquivos do projeto
3. Execute o arquivo
